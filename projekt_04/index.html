Semantic HTML elements

Tag Name Description
<header> Header contains introductory content or navigational aids
<nav> Navigation contains navigation links - menus, tables of contents, endexes
<main> Main represents the dominant content of the body of a document
<footer> Footer contains information about the author, copyright data or links
<section> Section represents a generic standalone section and it sould have a heading
<article> Article a self contained section, that is intended to be reusable (article, blog entry, product card, comment...)
<aside> Aside contains content only indirectly connected to the main document's content (sidebar, callout-box)
<figure> Figure self-contained content with a caption, specified with the <figcaption> element
<video> Video embeds a video player
<embed> Embed embeds external content, provided by an external application or a plugin
<audio> Audio embeds an audio player
